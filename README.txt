INFLC by Gideon
inflcmedia.com | @GiddDaKidd


This is INFLC, It's built around a grid of "image tiles" that are
set up to smoothly transition, and includes a number of neat effects (check out the menu!),
extra features, and all the usual stuff you'd expect. Hope you dig it!

Demo images* courtesy of Unsplash, a radtastic collection of CC0 (public domain) images
you can use for pretty much whatever.
